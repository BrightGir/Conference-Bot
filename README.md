## Conference bot
**Conference Bot** is a bot for Telegram designed to simplify the management of scientific seminars and conferences. This project was developed as a training project at SFU FMSh.

## 🌟 Features
- **Simple seminar creation:** You can easily set up seminars by specifying the title, date, time, and description. Perfect for organizing events quickly.
- **Stay notified:** The bot sends automatic reminders to participants about upcoming seminars, so nobody misses out.
- **Participant interaction:** Users can join seminars with a single click or leave them if plans change.
- **Organized by categories:** Seminars can be grouped into categories, making it easy for users to find events that interest them.
- **Admin tools:** Admins have access to special commands for managing events and overseeing participants.
